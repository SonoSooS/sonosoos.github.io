TiMidity++ 64bit

TiMidity++ is an ancient software synthesizer and sequenced music player.

Table of Contents

  • Changes
  • Downloads

Changes

  • Upgraded code to 64bit (original source code is incompatible with 64bit Windows)
  • Did some obvious optimizations
  • Added KDMAPI support
  • Removed arbitrary limits
  • Added 32bit signed integer and 32bit float output support

Bugs introduced

  • Stereo mode no longer works (the right channel is missing some things the left channel has)
  • Some UI elements no longer work. I can't really fix this without a rewrite, because the original 32bit code was already jank :/
The other bugs are sadly unfixed.
